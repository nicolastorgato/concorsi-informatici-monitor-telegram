# concorsi-informatici-monitor-telegram
🔍 Script che permette di monitorare i bandi di concorso pubblico per i profili di funzionario / specialista informatico nella Regione Veneto. Permette di ricevere notifiche giornaliere via Telegram con l'elenco dei concorsi attualmente aperti.

Sviluppato in Python, automatizzato con GitHub Actions.
